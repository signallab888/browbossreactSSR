Build a high-converting landing page for BrowBoss, a premium brow 
and beauty studio in La Jolla, San Diego, California.

═══════════════════════════════════════════════════════════
TARGET AUDIENCE
═══════════════════════════════════════════════════════════

American women, dual segment:
- Affluent clients (ages 35-60, disposable income, value luxury)
- Aspirational clients (ages 25-45, budget-conscious but willing 
  to invest in beauty)

The design must feel premium enough for affluent clients while 
including subtle accessibility signals (financing, starting prices, 
diverse testimonials) so price is never an objection.

═══════════════════════════════════════════════════════════
TECH STACK (MANDATORY)
═══════════════════════════════════════════════════════════

- Astro 4+ with static site generation (SSG)
- Tailwind CSS for styling
- Vanilla JavaScript only when needed
- NO React, NO Vue, NO heavy frameworks
- All HTML server-rendered for maximum SEO
- Static output deployable to Hostinger via FTP
- Build command: npm run build outputs to /dist

═══════════════════════════════════════════════════════════
DESIGN SYSTEM
═══════════════════════════════════════════════════════════

Aesthetic: Modern editorial luxury, black & white minimalist
Think: Vogue magazine meets boutique medical spa

Colors:
- Primary: Pure black (#0A0A0A)
- Background: Pure white (#FFFFFF)
- Accent: Warm cream (#F5F1EC) for section backgrounds
- Text secondary: Charcoal (#4A4A4A)
- NO gradients, NO bright colors, NO shadows except very subtle

Typography:
- Headings: Playfair Display (serif, editorial feel)
- Body: Inter (clean sans-serif)
- Accent/pricing: Inter medium weight
- Generous letter-spacing on uppercase labels

Layout:
- Max-width 1280px for content
- Generous whitespace (minimum 80px between sections desktop)
- Mobile-first responsive design
- Smooth scroll behavior
- Subtle fade-in animations on scroll only (no bouncing, no flashing)

═══════════════════════════════════════════════════════════
PERFORMANCE REQUIREMENTS
═══════════════════════════════════════════════════════════

- Lighthouse mobile score: 95+ across all metrics
- First Contentful Paint: under 1.5 seconds
- Largest Contentful Paint: under 2.5 seconds
- Total page size: under 1MB (excluding video)
- Images: WebP format with JPG fallback, lazy loaded below fold
- Hero video: compressed, max 2MB, with poster image
- Critical CSS inlined
- Fonts: preloaded, display=swap
- No render-blocking resources
- Minified HTML, CSS, JS

═══════════════════════════════════════════════════════════
ACCESSIBILITY (WCAG AA)
═══════════════════════════════════════════════════════════

- Proper ARIA labels on all interactive elements
- Keyboard navigable (tab order makes sense)
- Focus states visible
- Minimum 44x44px tap targets on mobile
- Color contrast ratios meeting AA standards
- Alt text on all images
- Form labels properly associated
- Skip-to-content link for screen readers
